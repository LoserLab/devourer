# Changelog

All notable changes to the Devourer skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-22

### Added
- Initial release of Devourer skill
- Support for React (TypeScript), Vue 3, and HTML+CSS component generation
- Color utilities script for variant generation and contrast checking
- Component patterns reference with detailed implementation examples
- CSS theme template with custom properties
- TypeScript type definitions template
- Support for 20+ component types across 4 categories:
  - Interactive: Button, Input, Select, Checkbox, Radio, Switch, Slider
  - Container: Card, Modal, Panel, Alert
  - Navigation: NavBar, Tabs, Breadcrumb, Pagination
  - Data Display: Table, Badge, Avatar, Progress
- WCAG AA/AAA contrast compliance checking
- Automatic color state variant generation (hover, active, disabled)
- Dark mode support
- Accessibility-first approach with ARIA attributes
- Integration patterns with Hexed color generator
- Comprehensive documentation and usage examples

### Features
- Parse color palettes in multiple formats (Hexed JSON, hex arrays, named colors)
- Generate production-ready components with variants and states
- Full TypeScript support with type safety
- Framework-specific best practices
- Semantic color mapping
- Responsive design patterns
- Focus management utilities
- Screen reader support

### Documentation
- Complete SKILL.md with workflow and best practices
- Component patterns reference guide
- README with usage examples
- MIT License
- Contributing guidelines
